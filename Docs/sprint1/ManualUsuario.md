<h1>Manual do Usuário</h1>

<h2 style="color: #0080c0;">- Objetivo do Sistema</h2>
<p>
O sistema tem como objetivo auxiliar o professor no planejamento de aulas,
permitindo visualizar as aulas do semestre, registrar conteúdos e verificar a carga horária.
</p>

<h2 style="color: #0080c0;">- Como Executar o Sistema</h2>
<ul>
  <li>Abrir o projeto na IDE (IntelliJ)</li>
  <li>Executar o projeto utilizando Maven (javafx:run)</li>
</ul>

<h2 style="color: #0080c0;">- Como Utilizar o Sistema</h2>
<ul>
  <li>Visualizar a lista de aulas com data e conteúdo</li>
  <li>Registrar o conteúdo de uma aula</li>
  <li>Distribuir conteúdos automaticamente ao longo do semestre</li>
  <li>Verificar se a quantidade de aulas corresponde à carga horária</li>
</ul>

<h2 style="color: #0080c0;">- Funcionalidades Disponíveis</h2>
<ul>
  <li>Exibição das aulas em ordem cronológica</li>
  <li>Registro de conteúdo por aula</li>
  <li>Distribuição automática de conteúdos</li>
  <li>Validação da carga horária da disciplina</li>
</ul>

<h2 style="color: #0080c0;">- Observações</h2>
<ul>
  <li>O sistema utiliza dados em memória</li>
  <li>Não há persistência em banco de dados nesta versão</li>
</ul>
